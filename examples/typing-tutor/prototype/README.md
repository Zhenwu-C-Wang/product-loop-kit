# Typing Tutor Prototype

This is a runnable static prototype for the Typing Tutor case study.

Open `index.html` directly in a browser, or serve the directory with any static server.

Run scoring tests:

```bash
node examples/typing-tutor/prototype/scoring.test.js
```

The prototype covers the first practice-flow slice:

- Adult difficulty control.
- Prompt display.
- Typing input.
- Accuracy and words-per-minute scoring.
- Result state.
- New prompt action.
